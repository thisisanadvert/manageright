# Manageright

One-page website for Right to Manage support services in Slough and Windsor.

## Live Site

https://thisisanadvert.github.io/manageright

## About

Manageright provides hands-on Right to Manage support for leaseholders in Slough, Windsor, and surrounding areas. The service helps leaseholders understand their options, plan the RTM process, and provides practical guidance throughout.

**Partnership:** Working in partnership with Manage.Management for operational expertise.

## Features

- Clean, modern one-page design
- Light/dark mode toggle
- Mobile responsive layout
- Clear call-to-action throughout
- Focus on local area (Slough & Windsor)

## Technology

- Single HTML file (no build process required)
- Vanilla JavaScript for theme switching
- CSS custom properties for theming
- Deployed via GitHub Pages

## Customization

To customize the site, edit `index.html`:

1. **Email address**: Replace `hello@manageright.solutions` with your actual email
2. **Booking link**: Replace `mailto:` links with Calendly or other booking tool URL
3. **Phone number**: Add to header or footer as needed
4. **Content**: Update copy for services, areas, or messaging

## Custom Domain

Once `manageright.solutions` is purchased:

1. Go to Settings → Pages in this repository
2. Add custom domain: `manageright.solutions`
3. Update DNS records at your domain registrar:
   - Add CNAME record pointing to `thisisanadvert.github.io`
   - Or follow GitHub's custom domain setup guide
